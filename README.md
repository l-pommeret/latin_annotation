# Annotation Latin UD – EvaLatin 2026

## Lancer l'interface

### Mac / Linux

```bash
git clone https://github.com/l-pommeret/latin_annotation.git
cd latin_annotation
./lancer.sh
```

### Windows

```bash
git clone https://github.com/l-pommeret/latin_annotation.git
cd latin_annotation
lancer.bat
```

### Prérequis

- Python 3 (déjà installé sur Mac/Linux, à installer sur Windows depuis [python.org](https://www.python.org/downloads/))
